Displays the PHP memory consuption in the footer of your admin backend.

== Description ==

When activated, this plugin will display the PHP memory consuption. e.g.: Load 6.41 of 128M. This is usefull when you've got limited PHP memory available, and are trying to keep track on how much you've got left. This plugin is based almost entirely on WP Overview (Lite).

== Installation ==

1. Upload the `php-mem-ind` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= I've found a bug. What do I do? =

= Will this plugin work with WP Overview (Lite)? =

No. If you are planning on using WP Overview (Lite), disable this plugin.
